QuickDB

Working with Databases has never been Easier! QuickDB aims to develop a persistence library that allow the user to write only the Data Model, and the library will manage all the operations between the Entities and the Database, without writing any line of code for Connection, Sql, etc. Where each operation involves only what you wanna do (save, modify, ...) and to what are you going to apply it (the object) .

Future Talks

NINJA-IDE, an IDE specially designed for Python

PyCon Uruguay (Nov, 2013)

NINJA-IDE is an IDE written in Python and for Python developers, the talk aims to show some cool features that IDE has, and how to take advantage of the potential of having our work tool written in Python to suit our needs.

Introduction to PyQt

PyCon Spain (Nov, 2013), PyCon Uruguay (Nov, 2013)

PyQt it's a binding to use the Qt framework from Python, this binding allow us to build incredible user interfaces with a lot of potential and simplicity. This talk aims to cover the fundamental aspects of developing PyQt Applications, showing some practical examples.

Making programs that understand programs with AST

PyCon Spain (Nov, 2013)

This talk aims to show how we can write code that "understand" code, using the AST (Abstract Syntax Trees) module, to analyze, parse and obtain metadata from any piece of code in order to provide tools such as: refactoring, code completion, lint checkers, etc.
